# Footprint&OrderFlow DataServer
Base on [orderflow](https://github.com/focus1691/orderflow), currently only supported Binance spot and 1m candle
## How to use
Git Clone
```
git clone git@github.com:404er/go-orderflow.git

cd go-orderflow

go mod tidy

go run main.go
```
